# Favorite Subdir

Right click a folder in the explorer to make it the default directory for new terminals.

## Usage

1. Right click on any folder in the Explorer.
2. Select **Set as Favorite Terminal Directory**.
3. Open a new terminal; it will start in that folder.

To clear the setting:
1. Open Command Palette (`Cmd+Shift+P`).
2. Run **Clear Favorite Terminal Directory**.
